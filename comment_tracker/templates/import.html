Import Guide
JSON Format (Recommended)

JSON files include project, batch, and comment data in one file. See the ingestion schema for the required structure.

CSV Format

CSV files contain only comment rows. Project and batch metadata must be provided via the form fields above.

Required Comment Fields
  • comment_number - Sequential number
  • comment_text - The comment content
  • severity - Major or Minor
  • category - Technical, Typo, Readability, FigTable, Format, Reference
  • status - Accepted, Accepted (modified), Noted, Rejected
Update Mode

If a batch (same project + revision) already exists, import will fail unless update mode is checked. Update mode deletes the existing batch data and replaces it.
