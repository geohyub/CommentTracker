데이터 임포트

구조화된 코멘트 데이터 업로드 (JSON 또는 CSV)

파일 업로드
여기에 파일을 드래그 앤 드롭하세요

또는 클릭하여 파일을 선택하세요. JSON, CSV 파일 지원. 여러 파일 동시 선택 가능.

선택된 파일:
코멘트 종류
CSV 메타데이터 (CSV 파일에 필요)
임포트 가이드
JSON 형식 (권장)

JSON 파일에는 프로젝트, 배치, 코멘트 데이터가 하나의 파일에 포함됩니다. 필수 구조는 인제스천 스키마를 참조하세요.

CSV 형식

CSV 파일에는 코멘트 행만 포함됩니다. 프로젝트 및 배치 메타데이터는 위의 폼 필드를 통해 제공해야 합니다.

필수 코멘트 필드
  • comment_number - 순번
  • comment_text - 코멘트 내용
  • severity - Major 또는 Minor
  • category - Technical, Typo, Readability, FigTable, Format, Reference
  • status - Accepted, Accepted (modified), Noted, Rejected
업데이트 모드

동일한 소스 파일이 이미 임포트된 경우에만 필요합니다. 같은 프로젝트의 다른 파일은 충돌 없이 자유롭게 업로드 가능합니다. 업데이트 모드는 기존 배치 데이터를 삭제하고 새 데이터로 교체합니다.
